# Business Scope

## Objective

Build an optimization-ready project foundation for production planning in a mixed manufacturing environment.

## Plant Context

- Two stable high-volume product families
- One unstable custom product family
- One service-parts stream that frequently interrupts weekly schedules

## Scope in Current Stage

- Establish a robust Python project structure
- Define configuration, data loading, validation, and reporting entry points
- Define model interfaces for LP/MIP placeholders
- Prepare test and documentation baseline

## Out of Scope in Current Stage

- Full optimization model implementation
- Solver tuning and advanced decomposition strategies
- ERP/MES integrations
